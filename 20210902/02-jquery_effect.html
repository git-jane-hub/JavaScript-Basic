jQuery Effect

제이쿼리에서는 화면전환 슬라이드 업다운
등의 다양한 효과 메서드 제공

show/hide Effect

show 와 hide는 선택요소를 노출시키거나 숨길 때 사용
show(시간)/hide(시간)
시간은 fast, normal, slow 등으로 입력하거나
혹은 밀리초(1/1000초)단위의 숫자로 입력하면 그 시간에 걸쳐 작동

toggle Effect

toggle은 선택요소가 숨겨져있다면, 보이도록
보인다면 숨기도록 만듦

slideDown/Up Effect

slideUp은 선택요소를 위로 말아올리며 숨기고
slideDown은 선택요소를 아래로 말아내려서 노출

fadeIn/fadeOut Effect

fadeOut은 선택요소를 천천히 사라지게 함
fadeIn은 선택요소를 천천히 노출되게 함

fadeTo Effect

fadeTo는 선택요소를 원하는 만큼만
보이거나 숨길 수 있음
